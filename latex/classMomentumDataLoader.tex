\doxysubsection{Detailed Description}
Class for loading momentum distribution data from text files. 

Class for loading and processing momentum distribution data from text files, specifically for analyzing N\texorpdfstring{$\ast$}{*} resonance within a \texorpdfstring{$^\wedge$}{\string^}3\+He nucleus. It supports converting momentum units from fm\texorpdfstring{$^\wedge$}{\string^}-\/1 to Ge\+V/c and normalizing probability densities.
