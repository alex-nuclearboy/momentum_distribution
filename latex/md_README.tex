\chapter{Nucleon and N\texorpdfstring{$\ast$}{*} Resonance Momentum Distribution Analysis}

This repository provides analytical tools for investigating the momentum distribution of nucleons within a deuteron as well as the momentum distribution of the N\texorpdfstring{$\ast$}{*}(1535) resonance within the \textsuperscript{3}He nucleus.

It uses different nucleon-\/nucleon interaction models, specifically the Paris \mbox{[}\href{https://doi.org/10.1016/0370-2693(81)90659-6}{\texttt{ 1}}\mbox{]} and the charge-\/dependent Bonn (CD-\/\+Bonn) \mbox{[}\href{https://doi.org/10.1103/PhysRevC.63.024001}{\texttt{ 2}}\mbox{]} potentials, to calculate and visually represent the nucleon momentum distribution in a deuteron. The process is streamlined by the software automatically processing model parameters provided in a JSON configuration file, resulting in both numerical data and graphical representations of the momentum distributions.

Additionally, the repository includes the tool for graphical representation of the momentum distribution of the N\texorpdfstring{$\ast$}{*} resonance within the \textsuperscript{3}He nucleus, considered as a bound system of the resonance and a deuteron. This analysis is based on the theoretical work of Kelkar {\itshape et al.} \mbox{[}\href{https://doi.org/10.1142/S0218301319500666}{\texttt{ 3}}\mbox{]}, \mbox{[}\href{https://doi.org/10.1016/j.nuclphysa.2020.121698}{\texttt{ 4}}\mbox{]}, in which the N\texorpdfstring{$\ast$}{*} momentum distribution is evaluated for various sets of coupling constants corresponding to binding energies E\textsubscript{N\texorpdfstring{$\ast$}{*}-\/d} = -\/0.\+33, -\/0.\+53, and -\/0.\+74 MeV. Comparing these distributions with the proton momentum distribution in \textsuperscript{3}He yields insights into exotic nuclear states\textquotesingle{} structure and dynamics.\hypertarget{md_README_autotoc_md1}{}\doxysection{\texorpdfstring{Key Features}{Key Features}}\label{md_README_autotoc_md1}

\begin{DoxyItemize}
\item {\bfseries{Deuteron Analysis\+:}} Uses Paris \mbox{[}\href{https://doi.org/10.1016/0370-2693(81)90659-6}{\texttt{ 1}}\mbox{]} and CD-\/\+Bonn \mbox{[}\href{https://doi.org/10.1103/PhysRevC.63.024001}{\texttt{ 2}}\mbox{]} nucleon-\/nucleon potentials for calculations of nucleon momentum distribution in a deuteron, derived from parameters specified in a JSON configuration file.
\item \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+N\texorpdfstring{$\ast$}{*}(1535) Resonance in \textsuperscript{3}He\+:\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} The tool converts N\texorpdfstring{$\ast$}{*} momentum values from fm\textsuperscript{-\/1} to Ge\+V/c and normalises the distribution, facilitating the visualisation of the N\texorpdfstring{$\ast$}{*} momentum distribution within the \textsuperscript{3}He nucleus. Additionally, it provides comparative visualisations of proton momentum distributions in \textsuperscript{3}He.
\end{DoxyItemize}\hypertarget{md_README_autotoc_md2}{}\doxysection{\texorpdfstring{Software Dependencies}{Software Dependencies}}\label{md_README_autotoc_md2}

Additionally, the project includes the {\ttfamily \doxylink{json_8hpp}{json.\+hpp}} header from the \href{https://github.com/nlohmann/json}{\texttt{ nlohmann/json}} library, located in the {\ttfamily include} directory for JSON parsing.\hypertarget{md_README_autotoc_md3}{}\doxysection{\texorpdfstring{Environment Setup}{Environment Setup}}\label{md_README_autotoc_md3}

\hypertarget{md_README_autotoc_md5}{}\doxysection{\texorpdfstring{Outputs}{Outputs}}\label{md_README_autotoc_md5}
The software produces text files detailing nucleon momentum distributions in a deuteron and text files with the N\texorpdfstring{$\ast$}{*} resonance\textquotesingle{}s momentum distributions in \textsuperscript{3}He, converted from from fm\textsuperscript{-\/1} to Ge\+V/ and normalised. These files are saved in the {\ttfamily data} folder.
