\doxysubsection{Detailed Description}
Implementation of the \doxylink{classMomentumDataLoader}{Momentum\+Data\+Loader} class for loading, processing, and plotting N\texorpdfstring{$\ast$}{*} resonance momentum distribution data. 

\begin{DoxyAuthor}{Author}
AK \href{mailto:alex.nuclearboy@gmail.com}{\texttt{ alex.\+nuclearboy@gmail.\+com}} 
\end{DoxyAuthor}
The \doxylink{classMomentumDataLoader}{Momentum\+Data\+Loader} class is designed for investigating momentum distributions of the N\texorpdfstring{$\ast$}{*}(1535) resonance within the context of the excited \texorpdfstring{$^\wedge$}{\string^}3\+He nucleus, interpreted as a system of the N\texorpdfstring{$\ast$}{*} and a deuteron. This class simplifies the tasks of loading, processing, and visualizing data across different binding energies, specifically for E\+\_\+\{N\texorpdfstring{$\ast$}{*}-\/d\} = -\/0.\+33 MeV, -\/0.\+53 MeV, and -\/0.\+74 MeV. Additionally, it processes proton momentum data within the \texorpdfstring{$^\wedge$}{\string^}3\+He nucleus based on two theoretical models.

The class streamlines the analysis workflow by\+:
\begin{DoxyItemize}
\item Loading data from text files for varied binding energies.
\item Converting momentum from fm\texorpdfstring{$^\wedge$}{\string^}-\/1 to Ge\+V/c, ensuring dataset consistency.
\item Normalizing probability densities for uniform comparison and analysis.
\item Storing processed data in output files for further analysis.
\item Creating a combined plot for all datasets, illustrating the momentum distributions of both the N\texorpdfstring{$\ast$}{*} resonance and the proton within the \texorpdfstring{$^\wedge$}{\string^}3\+He nucleus.
\end{DoxyItemize}
